# Telegram Бот

Простой Telegram бот для тестовых целей.

## Настройка

1. Создайте бота через [@BotFather](https://t.me/BotFather) в Telegram и получите токен.
2. Создайте файл `.env` в корне проекта с содержимым:
   ```
   TELEGRAM_BOT_TOKEN=ваш_токен_бота
   PORT=3000
   ```

## Локальный запуск

```bash
npm install
npm run bot
```

## Развертывание на Render

1. Создайте аккаунт на [Render](https://render.com/).
2. Подключите ваш Git-репозиторий.
3. Создайте новый Web Service, указав этот репозиторий.
4. Настройте переменные окружения:
   - `TELEGRAM_BOT_TOKEN` - токен вашего бота
5. Нажмите кнопку "Create Web Service".

После успешного деплоя ваш бот будет работать в режиме веб-хука.
