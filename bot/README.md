# Telegram Бот

Простой Telegram бот для тестовых целей.

## Настройка

1. Создайте бота через [@BotFather](https://t.me/BotFather) в Telegram и получите токен.
2. Создайте файл `.env` в корне проекта с содержимым:
   ```
   TELEGRAM_BOT_TOKEN=ваш_токен_бота
   PORT=3000
   ```

## Локальный запуск

```bash
npm install
npm run bot
```

## Развертывание на Render

1. Создайте аккаунт на [Render](https://render.com/).
2. Подключите ваш Git-репозиторий.
3. Создайте новый Web Service, указав этот репозиторий.
4. Настройте переменные окружения:
   - `TELEGRAM_BOT_TOKEN` - токен вашего бота
5. Нажмите кнопку "Create Web Service".

После успешного деплоя ваш бот будет работать в режиме веб-хука.

# Настройка бота в Telegram

Этот документ содержит подробные инструкции по настройке бота в Telegram для работы с мини-приложением "Разгильдяй".

## Основные команды BotFather

### Создание бота
1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям: укажите имя бота и его username (должен заканчиваться на "bot")
4. Получите токен бота и сохраните его в переменной окружения `TELEGRAM_BOT_TOKEN`

### Настройка меню команд
1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/setcommands`
3. Выберите вашего бота
4. Отправьте список команд в формате:
   ```
   start - Начать игру
   help - Показать помощь
   rules - Правила игры
   ```

### Настройка кнопки меню
1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/setmenubutton`
3. Выберите вашего бота
4. Введите текст для кнопки меню (например, "Играть")
5. Введите URL вашего мини-приложения (например, `https://telegram-razgildiay.onrender.com`)

## Создание мини-приложения

Для создания мини-приложения и получения Web App URL:

1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newapp`
3. Выберите вашего бота
4. Следуйте инструкциям:
   - Введите название приложения
   - Введите краткое описание
   - Загрузите иконку (должна быть квадратной, не менее 512x512 пикселей)
   - После создания, вы получите Web App URL

## Настройка кнопки для запуска мини-приложения

1. Откройте [@BotFather](https://t.me/BotFather)
2. Выберите вашего бота и отправьте команду `/mybots`
3. Нажмите "Bot Settings" -> "Menu Button" -> "Configure menu button"
4. Введите текст кнопки (например, "Играть в Разгильдяй")
5. Укажите URL вашего мини-приложения, полученный на предыдущем шаге

### Или настройте кнопку командой
```
/setmenubutton
```
